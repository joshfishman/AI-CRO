How to Use

  1. Set up the bookmarklet with your API base URL and editor key
  2. Navigate to the page you want to personalize
  3. Click the bookmarklet in your bookmarks bar
  4. Click on elements you want to personalize
  5. For each element, enter a prompt for AI to generate content
  6. Click "Save Config" when done
  7. Add the personalization loader script to your site

Example Prompts

  • Heading: "Write a catchy headline for a landing page selling eco-friendly water bottles"
  • Call to Action: "Write a compelling CTA button text for signing up for a newsletter"
  • Product Description: "Rewrite this product description to highlight benefits instead of features"
