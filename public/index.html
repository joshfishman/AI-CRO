AI CRO - Website Personalization

A powerful tool for website personalization and conversion rate optimization using artificial intelligence.

View on GitHub

How It Works

AI CRO dynamically personalizes your website content based on user types and behaviors, helping to increase conversion rates and improve user experience.

Quick Start

To integrate AI CRO into your website, add the following script to your site:

<script src="https://ai-cro-eight.vercel.app/personalization-loader.js" defer></script>

API Endpoints

  • /api/save-config - Save website personalization configuration
  • /api/get-config - Retrieve website personalization configuration
  • /api/personalize - Apply personalization to a webpage
  • /api/get-user-type - Determine user type for personalization
  • /api/selector-bookmarklet - Helper tool for selecting DOM elements
