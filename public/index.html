AI CRO - Website Personalization

A powerful tool for website personalization and conversion rate optimization using artificial intelligence.

View on GitHub

How It Works

AI CRO dynamically personalizes your website content based on user types and behaviors, helping to increase conversion rates and improve user experience.

Quick Start

To integrate AI CRO into your website, add the following script to your site:

<script src="https://ai-cro-eight.vercel.app/public/personalization-loader.js" defer></script>

How to Use

1. Set Up Personalization

  1. Select Elements to Personalize: Use the selector bookmarklet to identify elements on your page you want to personalize. 
    javascript:(function(){var script=document.createElement('script');script.src='https://ai-cro-eight.vercel.app/api/selector-bookmarklet.js';document.body.appendChild(script);})();

    Add this as a bookmark in your browser, then click it when on the page you want to personalize.

  2. Configure Selectors: For each element you select, provide:
    • A prompt describing what type of content should be generated
    • The default (original) content for reference
  3. Save Configuration: Use the API key to save your configuration: 
    fetch('https://ai-cro-eight.vercel.app/api/save-config', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'YOUR_EDITOR_API_KEY'
  },
  body: JSON.stringify({
    url: '/your-page-path',
    selectors: [
      {
        selector: 'h1.hero-title',
        prompt: 'Write a catchy headline for a landing page',
        default: 'Welcome to our service'
      },
      {
        selector: '.cta-button',
        prompt: 'Write a compelling call to action',
        default: 'Get Started'
      }
    ]
  })
});

2. Define User Types

AI CRO can personalize content based on user types. You can use the HubSpot integration or define your own user type logic.

The get-user-type API endpoint can be customized to return different user types based on your criteria.

3. Content Personalization

Once integrated, the AI CRO script will:

  1. Detect the current page URL
  2. Fetch the configuration for that page
  3. Determine the user type
  4. Generate personalized content using AI
  5. Replace the content on your page with personalized variants

4. Testing and Refinement

You can test different personalization strategies by:

  • Using the dry parameter in the personalize API to preview content
  • A/B testing different personalization approaches
  • Analyzing conversion improvements through your analytics platform

API Endpoints

  • /api/save-config - Save website personalization configuration
  • /api/get-config - Retrieve website personalization configuration
  • /api/personalize - Apply personalization to a webpage
  • /api/get-user-type - Determine user type for personalization
  • /api/selector-bookmarklet - Helper tool for selecting DOM elements
